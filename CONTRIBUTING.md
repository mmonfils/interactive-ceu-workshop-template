# Contributing to the Interactive CEU Workshop Template

Thank you for your interest in improving or utilizing this open-source instructional layout! This repository serves as a low-tech, mobile-friendly blueprint for academics, social work educators, and presenters who want to run highly structured, 90-minute interleaved CEU sessions.

## 📐 How to Use This Template for Your Project
1. **Fork the Repository:** Click the "Fork" button in the upper right corner to create a copy of this layout in your own GitHub account.
2. **Global Variable Replacement:** Scan through `README.md` and `EVALUATION.md` to find all items wrapped in double curly braces (e.g., `{{PRESENTATION_TITLE}}`). Replace them entirely with your specific project details.
3. **Configure GitHub Pages:** Go to your repository settings, navigate to the **Pages** menu, and select the `main` branch to host your presentation as a mobile-optimized public web page for your live session attendees.

## 🤝 Open-Source Contribution Rules
We welcome feedback to optimize this template for modern, mobile-first learning spaces:
* **Simplicity Constraint:** Do not submit pull requests adding JavaScript frameworks, advanced CSS libraries, or external dependencies. The primary goal is 100% clean, standard Markdown text that loads instantly on low-bandwidth cell phone browsers.
* **Pacing Modifications:** If you submit an alternative timeline layout, maintain a strict maximum of 15 continuous speaking minutes followed by interactive praxis breaks to preserve the core adult-learning pedagogical rhythm.
* **Ethical Guardrails:** Ensure that any edits made to the evaluation frameworks remain compliant with universal continuing education standard requirements.
